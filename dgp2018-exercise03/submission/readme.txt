Programming Assignment: "Marching Triangles"

Approach:
Analogous to the Marching Squares Algorithm we discriminate between predefined contouring patterns for
every triangle, stored in a lookup table. Afterwards we use linear interpolation, weighted by the
iso values on the involved corner points.
